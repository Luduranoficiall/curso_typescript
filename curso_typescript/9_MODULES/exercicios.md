# Exercícios — 9_MODULES

## Exercício 1
Crie dois arquivos TypeScript: math.ts (exporta funções somar e subtrair) e app.ts (importa e usa essas funções). Execute o app.ts.

## Exercício 2
Implemente um namespace Util com uma função para inverter strings. Use essa função em outro arquivo.

## Exercício 3
Crie um módulo que exporta um type alias para representar um usuário e uma função para criar usuários.

## Exercício 4 (Desafio)
Implemente um projeto simples que utiliza módulos ES6 para organizar funções matemáticas, tipos e classes. Teste a importação e uso de cada módulo.

---
Exercícios inéditos, práticos e otimizados para fixação de módulos, namespaces e organização de código em TypeScript.
