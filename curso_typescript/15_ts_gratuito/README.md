# 15_ts_gratuito — Conteúdo Extra/Gratuito

Neste módulo você encontra:
- Dicas de produtividade com TypeScript
- Ferramentas úteis
- Links para documentação e comunidades

## Dicas de produtividade
- Use o modo `strict` no tsconfig.json para maior segurança
- Aproveite o autocompletar do VS Code
- Sempre tipar funções e objetos

## Ferramentas úteis
- ESLint + TypeScript para padronização
- Prettier para formatação
- ts-node para rodar TS direto no terminal
