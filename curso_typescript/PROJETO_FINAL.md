# Projeto Final — Curso TypeScript Especialista


## Descrição
Desenvolva uma aplicação fullstack (API + Web) que integre todos os conceitos do curso, com desafios reais de mercado:
- Tipagem avançada e tipos condicionais
- Padrões de projeto (Repository, Singleton, Observer, Factory)
- Segurança ponta-a-ponta (criptografia, JWT, logs de auditoria, checklist automatizado)
- Testes automatizados (unitários, integração, E2E, acessibilidade)
- Performance, UX e monitoramento

## Requisitos Avançados
- API RESTful com autenticação JWT, refresh token seguro, criptografia AES, logs estruturados e auditoria
- Painel administrativo com controle de permissões (RBAC)
- Frontend React com atomic design, hooks customizados, lazy loading, memoização, acessibilidade, code splitting dinâmico
- Testes unitários, integração, E2E (Cypress/Playwright) e acessibilidade (axe)
- Deploy seguro (Docker, Vercel, Netlify ou AWS), CI/CD automatizado
- Integração com serviços externos (ex: envio de e-mail, logs, analytics)
- Monitoramento de erros e performance (Sentry, LogRocket, etc)

## Entregáveis
- Repositório organizado (monorepo ou multi-pacotes)
- Documentação clara (README, exemplos de uso, onboarding)
- Checklist de segurança, qualidade e UX
- Demonstração funcional (deploy online ou vídeo)

---
## Avaliação
- Cobertura de testes >90% (unitários, integração e E2E)
- UX validada com Lighthouse e axe
- Código limpo, modular e escalável
- Solução dos desafios dos módulos
- Criatividade e uso de padrões avançados

Use os exemplos e desafios dos módulos como base para o projeto final. Inove, documente e mostre domínio dos conceitos!
