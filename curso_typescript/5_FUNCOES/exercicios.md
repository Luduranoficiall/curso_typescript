# Exercícios — 5_FUNCOES

## Exercício 1
Implemente uma função que recebe um array de números e retorna um novo array com todos os valores triplicados.

## Exercício 2
Crie uma função que recebe um nome (string) e uma idade (number) e retorna uma mensagem formatada. Use parâmetros opcionais para a idade.

## Exercício 3
Defina um type para uma função que recebe dois números e retorna um boolean indicando se o primeiro é múltiplo do segundo. Implemente essa função.

## Exercício 4 (Desafio)
Implemente uma função de callback que recebe um array de strings e uma função de transformação. Aplique a transformação em todos os elementos e retorne o novo array.

---
Exercícios inéditos, práticos e otimizados para fixação de funções e tipos de funções em TypeScript.
