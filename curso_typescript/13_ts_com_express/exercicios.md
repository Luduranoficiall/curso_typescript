# Exercícios — 13_ts_com_express

## Exercício 1
Crie um endpoint GET /ping que retorna { ok: true } e tipa corretamente a resposta.

## Exercício 2
Implemente um endpoint POST /produto que recebe um produto tipado (nome, preço) e retorna o produto criado.

## Exercício 3
Adicione um middleware para logar todas as requisições, tipando corretamente o req e res.

## Exercício 4 (Desafio)
Implemente um endpoint DELETE /usuario/:id que remove um usuário de um array em memória, validando o tipo do id.

Exercícios inéditos, práticos e otimizados para fixação de TypeScript com Express.
