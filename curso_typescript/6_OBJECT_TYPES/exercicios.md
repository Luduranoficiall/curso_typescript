# Exercícios — 6_OBJECT_TYPES

## Exercício 1
Crie uma interface para representar um livro (título, autor, ano) e uma função que recebe um livro e imprime suas informações.

## Exercício 2
Implemente uma interface para um usuário com método autenticar(senha: string): boolean. Crie uma função que recebe um usuário e uma senha e retorna se a autenticação foi bem-sucedida.

## Exercício 3
Crie um type para representar um endereço (rua, número, cidade) e uma interface que estende esse type adicionando o campo estado.

## Exercício 4 (Desafio)
Implemente uma função que recebe um array de objetos que implementam uma interface Animal (nome, falar()) e imprime o nome e o som de cada animal.

---
Exercícios inéditos, práticos e otimizados para fixação de interfaces, types e métodos em TypeScript.
