# Exercícios — 4_OPERACOES_EM_ARR

## Exercício 1
Crie um array de números e utilize o método map para dobrar todos os valores.

## Exercício 2
Implemente uma função que recebe um array de strings e retorna um novo array apenas com as strings que possuem mais de 5 caracteres.

## Exercício 3
Crie uma tupla para representar um produto (nome: string, preço: number) e imprima seus valores.

## Exercício 4 (Desafio)
Crie uma função que recebe um array de números e retorna a soma de todos os valores pares usando reduce.

---
Exercícios inéditos, práticos e otimizados para fixação de arrays e tuplas em TypeScript.
