# 2_FUNDAMENTOS_DO_TS — Fundamentos do TypeScript

Neste módulo você vai aprender:
- Tipos primitivos (string, number, boolean, null, undefined, any)
- Inferência de tipos
- Variáveis e constantes
- Comentários
- Operadores básicos

## Tipos primitivos
```ts
let nome: string = 'Maria';
let idade: number = 25;
let ativo: boolean = true;
let valor: any = 'pode ser qualquer coisa';
```

## Inferência de tipos
```ts
let cidade = 'São Paulo'; // string
let ano = 2025; // number
```

## Variáveis e constantes
```ts
const pi: number = 3.14;
let mutavel: string = 'pode mudar';
```

## Comentários
```ts
// Comentário de linha
/* Comentário de bloco */
```

## Operadores básicos
```ts
let soma = 2 + 3;
let texto = 'Olá' + ' mundo';
```
