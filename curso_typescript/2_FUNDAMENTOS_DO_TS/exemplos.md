# Exemplos práticos — 2_FUNDAMENTOS_DO_TS

## Exemplo 1: Tipos básicos
```ts
let nome: string = 'João';
let idade: number = 30;
let ativo: boolean = true;
let valor: any = 'pode ser qualquer coisa';
```

## Exemplo 2: Inferência de tipos
```ts
let cidade = 'São Paulo'; // string
let ano = 2025; // number
```

## Exemplo 3: Variáveis e constantes
```ts
const pi: number = 3.14;
let mutavel: string = 'pode mudar';
mutavel = 'mudou!';
```

## Exemplo 4: Operadores
```ts
let soma = 2 + 3;
let texto = 'Olá' + ' mundo';
```

---
Esses exemplos podem ser usados como base para os exercícios do módulo.
