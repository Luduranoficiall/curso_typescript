# Exercícios e Desafios Avançados de Arquitetura

## Microserviços
- Separe a aplicação em 2 serviços: API de usuários e API de pagamentos.
- Implemente comunicação via REST ou mensageria (ex: RabbitMQ).
- Desafio: autenticação JWT compartilhada entre serviços.

## Monorepo Avançado
- Estruture um monorepo com workspaces (Yarn/NPM/PNPM).
- Compartilhe tipos, utilitários e middlewares entre pacotes.
- Desafio: CI/CD para build/test/deploy de múltiplos pacotes.

## Testes E2E e CI/CD
- Implemente testes E2E com Cypress ou Playwright.
- Configure pipeline CI/CD (GitHub Actions, GitLab CI, etc).
- Desafio: deploy automatizado em múltiplos ambientes (dev, staging, prod).

## Logs, Monitoramento e Métricas
- Implemente logs estruturados (Winston, Pino, Morgan).
- Integre monitoramento (Sentry, LogRocket, Prometheus).
- Desafio: crie dashboards de métricas e alertas customizados.

---
Use estes desafios para elevar seu projeto final ao nível de especialista!
