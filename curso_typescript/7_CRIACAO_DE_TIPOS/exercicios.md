# Exercícios — 7_CRIACAO_DE_TIPOS

## Exercício 1
Crie um type alias para representar um pedido (id, produto, quantidade, status: 'aberto' | 'fechado').

## Exercício 2
Implemente uma função que recebe um objeto do tipo pedido e retorna uma string formatada com as informações.

## Exercício 3
Utilize o utility type Partial para criar uma função que atualiza parcialmente um pedido.

## Exercício 4 (Desafio)
Implemente uma função que recebe um array de pedidos e retorna apenas os que estão com status 'aberto'.

---
Exercícios inéditos, práticos e otimizados para fixação de criação e manipulação de tipos em TypeScript.
