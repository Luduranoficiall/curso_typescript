# Exercícios — 12_todo_react

## Exercício 1
Implemente a função para adicionar uma nova tarefa, garantindo que o texto não seja vazio e que o id seja único.

## Exercício 2
Adicione a funcionalidade de marcar uma tarefa como concluída, alterando o estado de concluído.

## Exercício 3
Implemente a remoção de tarefas, garantindo que o array de tarefas seja atualizado corretamente.

## Exercício 4 (Desafio)
Adicione um filtro para exibir apenas tarefas concluídas ou pendentes, usando tipagem em todas as funções.

Exercícios inéditos, práticos e otimizados para fixação de React com TypeScript em projetos reais.
