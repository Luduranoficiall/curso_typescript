
# Curso Completo de TypeScript

Este curso cobre do básico ao avançado, com exemplos práticos, projetos reais e exercícios inéditos. Todo o conteúdo foi cuidadosamente estruturado para proporcionar a melhor experiência de aprendizado e prática profissional em TypeScript.

## Sumário

### Módulo 1: Introdução ao TypeScript
- O que é TypeScript?
- Instalação e configuração do ambiente
- Primeiro código em TypeScript
- Compilando para JavaScript
- Tipos primitivos

### Módulo 2: Tipos e Variáveis
- Tipos básicos
- Inferência de tipos
- Tipos personalizados (type e interface)
- Union Types e Intersection Types
- Type Aliases

### Módulo 3: Funções
- Tipagem em funções
- Parâmetros opcionais e default
- Funções anônimas e arrow functions
- Funções como tipos

### Módulo 4: Objetos e Classes
- Objetos tipados
- Classes e construtores
- Modificadores de acesso
- Herança e polimorfismo
- Getters e Setters

### Módulo 5: Avançando com TypeScript
- Generics
- Decorators
- Namespaces e Modules
- Type Assertions
- Utility Types

### Módulo 6: TypeScript na Prática
- Trabalhando com bibliotecas JavaScript
- TypeScript com Node.js
- TypeScript com React
- Configuração avançada do tsconfig.json

### Módulo 7: Desafios e Exercícios
- Exercícios práticos por módulo
- Projetos finais

---

Cada módulo terá exemplos práticos e exercícios para fixação.

## Como usar este curso
1. Siga a ordem dos módulos.
2. Leia as explicações e execute os exemplos.
3. Resolva os exercícios propostos.
4. Consulte a documentação oficial sempre que necessário.

Bom estudo!
